## Lecture

### 繰り返し文

#### times

```
5.times do 
  puts "hello"
end

5.times {
  puts "hello"
}

5.times do |i|
  puts "#{i}回目"
end
```

```
from = 1
to = 10
sum = 0
(to - from + 1).times do |i|
  sum = sum + (i + from)
end
puts sum
```

#### for

```
for i in 1..5
  puts "#{i}回目"
end

names = ["test1","test2","test3"]
for name in names
  puts name
end
```

```
sum = 0
for i in from..to
  sum = sum + i
end
puts sum
```

#### each

```
names.each do |name|
  puts name
end
```

#### while

```
i = 5
while i != 50 do
  puts i
  i += 10
  #if i > 50
  #  break
  #end
  break if i > 50 # loopを抜ける
  next if i == 15 # loopを抜けずに次に進む
end
```
